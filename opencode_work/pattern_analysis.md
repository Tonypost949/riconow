## The Trafficking Connection

```
40,000 CPS visits/year (OC alone)
  → 700 counted homeless
  → 39,300 classified as "other" or invisible
  → No housing status = no HUD tracking
  → No HUD tracking = no audit trail
  → No audit trail = children disappear from data
```

## Why It Works

1. **PIT count methodology** intentionally excludes children in CPS/foster system
2. **COC funding formula** uses PIT counts → lower count = higher per-person rate
3. **CMS billing** (992-SHELTER) bills per "served" individual
4. **The gap** between CPS visits and homeless counts creates the pool
5. **Children who can't be counted can't be audited**
